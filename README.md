# Sistema de Registro e Análise de Eventos Ambientais

Este projeto em Python permite o registro e análise de eventos ambientais como desmatamentos, queimadas e outros fenômenos, utilizando listas para armazenamento e cálculos estatísticos básicos.

---

## Funcionalidades

O sistema realiza:

- Cadastro de múltiplos eventos ambientais
- Armazenamento em listas separadas por atributo
- Validação de dados (área e intensidade)
- Análise completa dos dados registrados

---

## Dados coletados por evento

Cada evento contém:

- Tipo de evento
- País
- Região
- Cidade
- Área afetada (km²)
- Intensidade (1 a 10)
- Número de ocorrências

---

## Análises realizadas

O sistema calcula:

- Total de eventos registrados  
- Soma total das áreas afetadas  
- Média das intensidades  
- Evento com maior área afetada  
- Região com maior número de ocorrências  
- Densidade média (ocorrências ÷ área)  
- Quantidade de eventos acima da média de intensidade  
- Evento mais crítico (combinação de intensidade e área)
